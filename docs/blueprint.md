# **App Name**: EthioHire

## Core Features:

- AI Application Assistant: Utilizes generative AI as a tool to produce tailored CVs and cover letters based on user profile and a given job description, also provides a match score.
- LinkedIn Job Aggregator: Allows users to paste LinkedIn job URLs to automatically extract job details (title, company, description) and save them to the database.
- User Profile Management: Enables users to create and manage their profile including skills, experience, education, and CV (PDF) uploads, which feed into the AI assistant.
- Application Tracking Dashboard: Displays all saved jobs with their application status (saved, applied, interview, rejected), providing search and filter capabilities.
- Streamlined Application Helper: Provides a step-by-step application checklist, autofill suggestions, and copy-ready answers to common application questions to expedite the application process.
- Admin Data Overview: A simplified panel for administrators to view registered users and basic aggregated job data, focused on oversight rather than full moderation for MVP.

## Style Guidelines:

- Primary color: A deep, professional indigo blue (#3B3399) conveying trust and ambition, appropriate for a career-focused application.
- Background color: A very light, subtle grey with a hint of purple (#F1F0F4), providing a clean, modern canvas that softly complements the primary hue.
- Accent color: A vibrant, clear sky blue (#528DE0) to highlight interactive elements, calls to action, and indicate success, providing dynamic contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a contemporary, tech-savvy, and distinctive appearance. Body font: 'Inter' (sans-serif) for excellent readability and a modern, objective feel in longer text blocks.
- Employ a consistent set of crisp, modern, and professional vector icons to enhance usability and reflect the app's clean aesthetic.
- Implement a clear dashboard layout featuring a persistent sidebar for navigation, ensuring the design is fully mobile-responsive across all device sizes for an optimal user experience.
- Incorporate subtle and functional animations for feedback on user interactions (e.g., button clicks, job saving) and transitions between dashboard views, avoiding distraction.
